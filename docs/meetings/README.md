# Meetings

This folder stores sanitized Maker Compass meeting artifacts.

## Folders

- `sanitized/`: cleaned meeting notes derived from transcripts or meeting notes.
- `decisions/`: decisions made in meetings.

Raw transcripts do not belong here. If raw transcript files must exist locally, keep them under `.private/raw-transcripts/meetings/`, which is ignored by git.

Before processing meeting notes, read `docs/operating-system/transcript-sanitization-protocol.md` and `docs/operating-system/meeting-notes-format.md`.
